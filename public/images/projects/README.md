# Project Screenshots

Untuk menambahkan screenshots ke halaman detail project, letakkan file gambar di folder `public/images/projects/` dengan format:

- `find-house-1.png`, `find-house-2.png`, `find-house-3.png` (untuk Find House App)
- `quranwise-1.png`, `quranwise-2.png`, `quranwise-3.png` (untuk Quranwise App)
- `presence-1.png`, `presence-2.png`, `presence-3.png` (untuk Presence GetX)
- `dompetku-1.png`, `dompetku-2.png`, `dompetku-3.png` (untuk Dompetku App)

## Rekomendasi:

- Format: PNG atau JPG
- Rasio: 9:16 (portrait untuk mobile apps)
- Ukuran: Maksimal 1MB per gambar
- Kualitas: 80-90% untuk optimasi web

Setelah menambahkan gambar, halaman detail project akan otomatis menampilkan screenshots yang sebenarnya.
